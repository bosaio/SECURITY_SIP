## üîß Troubleshooting

### If Sanity Studio Won't Load
- Check that it's running: `npm run dev` in the root directory
- Verify port 3333 is available
- Check browser console for errors

### If Content Won't Save
- Ensure all required fields are filled
- Check for validation errors
- Try refreshing the page

### If Website Won't Show Posts
- Verify Next.js client is running: `npm run dev` in client directory
- Check that Sanity queries are working
- Verify environment variables are set
